### Triangle problem statement from IF 
```sql
SELECT x,y,z,
CASE WHEN x+y <= z THEN 'No'
    WHEN y+z <= x THEN 'No' 
    WHEN z+x <= y THEN 'No' 
    ELSE 'Yes' END as `triangle`
FROM triangle;
```
